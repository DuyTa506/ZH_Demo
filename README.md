# Chuyển đổi giọng nói thành văn bản tiếng Trung - Demo Streamlit

## Cài đặt

Để cài đặt các thư viện cần thiết, chạy lệnh sau:

```bash
pip install -r requirements.txt
```

## Sử dụng

### Trước khi chạy demo, bạn cần tải các mô hình cần thiết. Sử dụng script download.bash để thực hiện việc này.

```bash
bash ./download.bash
```

## Chạy demo

### Để chạy demo, sử dụng script run.bash. Mặc định, demo sẽ chạy trên cổng 5000. Tuy nhiên, bạn có thể cấu hình cổng bằng cách sử dụng tùy chọn --port.

```bash
bash ./run.bash --port <port_number>
```
Nếu bạn không cấu hình cổng, mặc định sẽ là cổng 5000.
